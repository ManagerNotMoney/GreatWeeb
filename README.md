# 🌿 GreatWeeb — каннабис-дополнение для Minecraft

**GreatWeeb** — плагин для Paper 1.20.4+ (и совместимых форков), добавляющий в игру полноценную систему выращивания, переработки и употребления каннабиса.  
Два штамма (сатива и индика), гашиш, спайс, медицинские продукты, бонги, сушилки, удобрения и многое другое, уникальными эффектами и развитой механикой передозировки.

> ⚠️ Плагин предназначен для взрослых серверов с соответствующими правилами. Используйте на свой страх и риск.

---

## 🚀 Возможности

- **Выращивание**: посадка семян на грядку, уход за растениями (полив, удобрение, увлажнение льдом), сбор соцветий ножницами.
- **Переработка**: сушка соцветий в сушилке (бочка) → бошка → медицинская бошка + гашишное масло; измельчение ножницами; крафт брикетов и паков.
- **Изготовление**: косяки, бонги, удобрения, гашиш, гашишное масло, гашишный пирог, спайс, канна-брауни, вкусное печенье, медицинская жвачка.
- **Употребление**: курение косяков и бонгов (с зажигалкой), поедание продуктов — с отложенными эффектами, уникальными для каждого сорта.
- **Система передозировки**: отслеживание частоты и смешивания сортов; негативные эффекты при превышении порога (тошнота, слепота, отравление, иссушение для спайса).
- **Визуальные эффекты**: зелёный дым над игроком после употребления, частицы при курении.
- **Бонг**: многоразовое устройство, заряжаемое разными веществами; курение с зажигалкой; возможность разрядить.
- **Сушка**: блок-сушка (бочка) с GUI и прогрессом; переработка соцветий → бошки, бошек → мед. бошки + масло, измельчённой травы (×3) → мед. бошки + масло.
- **Брендирование брикетов**: нанесение названия бренда с помощью переименованной бумаги.
- **Увлажнение растений**: лёд (обычный, плотный, синий) защищает куст от засыхания.
- **Команды для администраторов**: выдача любых предметов, управление эффектами, очистка истории курения.

## 📦 Установка

1. Скачайте последний релизный `.jar` плагина из [Releases](https://github.com/ManagerNotMoney/GreatWeeb/releases).
2. Поместите файл в папку `plugins/` сервера.
3. Перезапустите сервер (или выполните `/reload confirm`).
4. Плагин не требует настройки — всё работает «из коробки».

**Системные требования:**
- Сервер: Paper 1.20.4+ (или форк с поддержкой Adventure и PersistentDataContainer)
- Java 17 или новее

## 🌱 Выращивание

1. **Посадка**: возьмите семена (`сатива` / `индика`) и используйте на вспаханной грядке. Блок автоматически превратится в пшеницу, привязанную к сорту.
2. **Уход**:
   - **Полив**: если куст засох (статус «Засохло» при клике), используйте ведро воды на нём.
   - **Удобрение**: крафтовое удобрение (`/gw fertilizer`) можно применить **только на только что посаженном кусте** (возраст 0). Увеличивает урожай и скорость роста.
   - **Костная мука**: ускоряет рост, но не даёт бонусов к урожаю.
   - **Увлажнение**: используйте **лёд** (любой) на растении, чтобы на время защитить его от засыхания. Повторное применение обновит защиту.
3. **Сбор урожая**: когда пшеница полностью вырастет, возьмите **ножницы** и кликните правой кнопкой по растению. Получите 2–4 соцветия (или больше, если куст удобрен). После сбора куст возвращается в начальную стадию роста.
4. **Разрушение куста**: при ломании выпадает одно семя и одно соцветие (если куст созрел). При вытаптывании грядки также выпадает семя.

## 🔥 Сушка (Drying Rack)

**Сушка** — это кастомный блок (бочка), создаваемый по рецепту (см. раздел «Рецепты»).  
Поставьте бочку, откройте инвентарь правым кликом.

Внутри интерфейса:
- **Слот 1** (вход) – сырьё.
- **Слот 2** – индикатор прогресса.
- **Слот 3** – основной выход.
- **Слот 4** – дополнительный выход (гашишное масло при обработке бошек/измельчённой травы).

**Процессы сушки** (каждые несколько секунд прибавляется прогресс, требуется ~900 условных тиков на операцию):
- Соцветие → Бошка (сорт сохраняется).
- Бошка → Медицинская бошка (слот 3) + Гашишное масло (слот 4).
- Измельчённая марихуана ×3 → Медицинская бошка + Гашишное масло.

Готовые предметы автоматически перекладываются в выходные слоты, а если там занято – падают рядом с сушкой.

## ✂️ Измельчение

Возьмите в **основную руку** бошку (сатива/индика), в **дополнительную** – ножницы, и кликните правой кнопкой.  
Одна бошка превращается в **2 измельчённой марихуаны**. Соцветие даёт **3 единицы** измельчённой травы. Ножницы теряют 5 прочности.

## 🧱 Брикеты, паки и брендирование

- **Брикет** — крафтится из **8 бошек + 1 бумага** (бесформенный рецепт).  
- **Пак** — крафтится из 9 брикетов (квадрат 3×3 в верстаке).  
- **Распаковка**: ПКМ брикетом в воздухе/блоке (не на верстаке) разламывает его обратно в 8 бошек. Пак разламывается в 9 брикетов.  
- **Брендирование**: возьмите в основную руку брикет, в дополнительную – **переименованную на наковальне бумагу** (название бренда), и нажмите Shift+ПКМ. Название появится в описании брикета. Бумага расходуется.

## 💨 Употребление

### Косяк (Joint)
- Основная рука: косяк (сатива/индика/медицинский).
- Дополнительная рука: зажигалка (Flint and Steel).
- ПКМ — зажигаете косяк. Эффекты наступают **через 20 секунд** (для медицинского – через 10 секунд).

**Эффекты:**
- *Сатива*: Скорость II (30 с), Регенерация I (7 с), Голод II (60 с) + зелёный дым.
- *Индика*: Замедление I (20 с), Ночное зрение (20 с), Голод I (40 с) + дым.
- *Медицинский*: Регенерация I (10 с), Замедление I (20 с), Голод I (5 с).

### Бонг
- **Зарядка**: держите в руке пустой бонг и нажмите Shift+ПКМ, имея в левой руке подходящий наполнитель (бошка, гашиш, спайс, медицинская бошка). Бонг сохранит тип содержимого.
- **Курение**: заряженный бонг в основной руке + зажигалка в левой → ПКМ. Эффекты наступают **через 10 секунд** и сильнее, чем от косяка.
- **Разрядка**: ПКМ с бонгом, когда левая рука пуста, вернёт наполнитель обратно.

**Эффекты бонга (примеры):**
- *Сатива*: Скорость, Регенерация, Голод (усиленные).
- *Индика*: Замедление, Ночное зрение, Голод.
- *Гашиш*: Замедление III, Регенерация, Слепота, Голод III.
- *Спайс*: Скорость, Голод III, Прыгучесть, зелёный дым.
- *Медицинская*: лёгкая Регенерация, Замедление, Голод.

### Еда
- **Канна-брауни**, **Вкусное печенье**, **Гашишный пирог** — клик правой кнопкой съедает предмет и через задержку применяет эффекты.  
  *Канна-брауни*: Регенерация II (40 с), Замедление I (30 с), Голод III (80 с), Тошнота (10 с).  
  *Пирог*: Регенерация I (40 с), Замедление I (60 с).  
  *Печенье*: Регенерация I (15 с), Тошнота (10 с).

### Медицинская жвачка
Съедается ПКМ, **удаляет зелёный дым** (запах) с игрока.

## ☠️ Передозировка

Система отслеживает число принятых веществ за последние **90 секунд** и факт смешивания сортов.

- **Пороги**: для большинства сортов — **6** доз; для спайса — **3**; для канна-брауни — **4**.
- **Смешивание** (например, сатива + индика) мгновенно вызывает передозировку.

**Симптомы передозировки:**
- Тошнота III (30 с)
- Слепота (15 с)
- Слабость II (60 с)
- Отравление II (10 с)
- Замедление III (20 с)
- Для спайса дополнительно Иссушение II (120 с)

После передозировки история курения сбрасывается, и повторные дозы временно не вызывают новых передозировок, но высвечивают сообщение «Мне становится ещё хуже...».

## 🧪 Медицинские продукты

- **Медицинская бошка** – получается сушкой обычной бошки или крафтом.
- **Медицинский косяк** – даёт слабые лечебные эффекты, не вызывает передозировку.
- **Медицинская жвачка** – крафтится из медицинской бошки, слизи и сахара; убирает запах.

## 📜 Рецепты (основные)

| Предмет | Ингредиенты |
|---------|-------------|
| **Семена** (сатива/индика) | 6 соцветий + костная мука (бесформенный) |
| **Косяк** | 2 бумаги + соответствующая бошка (вертикально) |
| **Брикет** | 8 бошек + 1 бумага (бесформенный) |
| **Пак** | 9 брикетов (3×3) |
| **Гашиш** | Бошка в печи (сатива — 250 тиков, индика — 200) |
| **Гашишное масло** | Гашиш в печи (300 тиков) |
| **Гашишный пирог** | 2 гашишного масла + гашиш + 3 пшеницы (бесформенный) |
| **Спайс** | Светокаменная пыль + 2 измельчённой травы + гашишное масло (бесформенный) |
| **Бонг** | Стеклянная бутылка + палка + огниво (бесформенный) **или** стеклянная бутылка + стержень ифрита |
| **Сушка (бочка)** | 6 досок (верх и низ ряда) + 2 нити (средний ряд по бокам) – shaped |
| **Удобрение** | Костная мука + гнилая плоть + измельчённая трава (бесформенный) |
| **Канна-брауни** | 2 пшеницы + какао-бобы + сахар + гашишное масло (бесформенный) |
| **Вкусное печенье** | 2 пшеницы + медицинская бошка → 2 печенья (бесформенный) |
| **Медицинская жвачка** | Медицинская бошка + слизь + сахар (бесформенный) |
| **Копчение бошки** | Соцветие в коптильне → бошка (50% опыт, 250 тиков) |

> Все рецепты регистрируются автоматически при загрузке плагина.

## 🛠️ Для разработчиков

Плагин написан на Java с использованием Paper API. Архитектура:
- `GreatWeeb` — главный класс, регистрация событий, менеджеров.
- `StrainItemsImpl` — общая логика предметов сативы и индики.
- `RecipeManager` — все крафты и печные рецепты.
- `OverdoseListener` — отслеживание и наказание за передоз.
- `DelayedEffectManager`, `SmokeEffectManager` — управление эффектами.
- `DryingRackManager` — циклическая обработка сушилок.
- Различные `*Listener` — обработка взаимодействий игроков.

Плагин **не использует конфигурационные файлы**; все параметры (пороги, длительности, множители) заданы в коде. При необходимости их можно вынести в `config.yml` (pull request’ы приветствуются).
