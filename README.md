# 🌿 GreatWeeb — дополнение для сервера с тематикой каннабиса

GreatWeeb — это плагин для серверов Minecraft (Paper 1.20+), добавляющий полную систему выращивания, переработки и употребления каннабиса. Включает два штамма (Сатива и Индика), а также гашиш, спайс, медицинские продукты, бонги, сушилки и многое другое. Все предметы имеют кастомные модели, рецепты крафта и уникальные эффекты.

> **Внимание:** Плагин предназначен для взрослых серверов с соответствующими правилами. Используйте на свой страх и риск.

## 🚀 Возможности

- **Выращивание** — посадите семена Сативы или Индики, ухаживайте за растениями (полив, удобрение), собирайте соцветия ножницами.
- **Переработка** — сушите соцветия в сушилке, получайте бошки, измельчайте их ножницами, прессуйте в брикеты и паки.
- **Изготовление** — крафтите косяки, бонги, удобрение, гашиш, гашишное масло, гашишный пирог, спайс, канна-брауни, вкусное печенье.
- **Употребление** — курите косяки или бонги с различными эффектами (скорость, замедление, регенерация, слепота, голод). Поддерживаются медицинские сорта.
- **Система передозировки** — при частом употреблении или смешивании сортов наступают негативные эффекты (тошнота, слабость, отравление).
- **Задержка эффектов** — эффекты наступают не мгновенно, а через некоторое время после курения.
- **Визуальные эффекты** — зелёный дым над игроком, частицы дыма при курении.
- **Бонг** — многоразовое устройство, которое можно заряжать бошкой/гашишем/спайсом, использовать с зажигалкой и разряжать.
- **Сушка** — специальный блок (бочка), в котором соцветия превращаются в бошки, а бошки — в медицинские бошки (+ гашишное масло).
- **Брендирование** — можно наклеить на брикет бумагу с названием бренда (любым переименованным листом).
- **Команды администратора** — выдача любых предметов, управление эффектами.

## 📦 Установка

1. Скачайте последний релиз плагина.
2. Поместите `.jar` в папку `plugins` вашего сервера.
3. Перезапустите сервер.
4. Плагин не требует конфигурации — все предметы и рецепты регистрируются автоматически.

> **Требования:**  
> - Сервер на базе Paper 1.20.4+ (или другой с поддержкой Adventure и Persistent Data Container).  
> - Java 17 или выше.

Все команды используют основной алиас `/gw`. Требуется разрешение `greatweeb.admin`.

## 📖 Основные механики

### 🌱 Выращивание

1. Получите семена (`/gw sativa seed`).
2. Посадите на грядку (правый клик на земле ножницами? Нет — просто посадите семя как обычно, оно превратится в пшеницу с сохранением сорта).
3. Ухаживайте:
   - **Полив** — если растение засохло (статус «Засохло»), используйте ведро воды.
   - **Удобрение** — используйте удобрение на только что посаженном растении (ускоряет рост и увеличивает урожай).
   - **Костная мука** — ускоряет рост (но не даёт бонуса урожая).
4. Когда пшеница полностью вырастет, возьмите **ножницы** и нажмите ПКМ на растении — соберёте соцветия (2–4 шт., +1–2 если удобрено). Растение сбросит возраст до 0 и сможет расти снова.

> Соцветия — это полуфабрикат, который нужно высушить.

### 🔥 Сушка (Drying Rack)

1. Скравьте **сушку** (`/gw bong`? нет, сушка крафтится из досок и верёвок — рецепт есть в коде). *Примечание: сушка — это бочка (barrel).*
2. Поставьте её на землю, откройте ПКМ.
3. Положите в слот №1 (вход) соцветия. Через некоторое время в слоте №3 появится бошка.
4. Положите обычную бошку — получите медицинскую бошку (слот №3) и гашишное масло (слот №4).
5. Положите 3 измельчённой марихуаны — получите медицинскую бошку + масло.

Прогресс отображается в центральном слоте.

### ✂️ Измельчение

Возьмите в основную руку бошку (любую), в дополнительную — ножницы. Нажмите ПКМ — бошка превратится в 2 измельчённой марихуаны. Ножницы тратят 5 прочности.

### 🧱 Брикеты и паки

- **Брикет** крафтится из 8 бошек + бумага (форма: 8 бошек вокруг бумаги). Разламывание брикета (ПКМ) даёт 8 бошек.
- **Пак** крафтится из 9 брикетов, разламывается обратно на 9 брикетов.
- **Брендирование** — при крафте или удерживая брикет и переименованную бумагу (+Shift+ПКМ) можно нанести название бренда (лоре).

### 💨 Употребление

#### Косяк (joint)
- Возьмите косяк в основную руку, зажигалку (Flint and Steel) в дополнительную.
- Нажмите ПКМ — зажжёте косяк. Эффекты наступят через 20 секунд.
- Возможные эффекты:
  - **Сатива:** скорость II, регенерация I, голод II.
  - **Индика:** замедление I, ночное зрение, голод I.
  - **Медицинский:** регенерация I, замедление I, голод I (без передозировки).

#### Бонг
- Крафтится из стеклянной бутылки + палка + огниво (или стеклянная бутылка + стержень ифрита).
- Зарядите бонг (Shift+ПКМ) с бошкой/гашишем/спайсом в дополнительной руке.
- Для курения: заряженный бонг + зажигалка во второй руке → ПКМ.
- Эффекты более сильные, наступают через 10 секунд.
- После курения бонг опустошается.

#### Еда
- **Канна-брауни**, **гашишный пирог**, **вкусное печенье** — съедаются как обычная еда (ПКМ). Эффекты с задержкой.

### ☠️ Передозировка

Плагин отслеживает употребление в течение последних 90 секунд. При превышении порога (по умолчанию 6 затяжек, для спайса — 3) или смешивании разных сортов наступает передозировка:
- Тошнота III, слепота, слабость II, отравление II, замедление III.
- Для спайса добавляется иссушение II.
- После передозировки игрок временно не может получать эффекты от веществ.

### 🧪 Медицинские продукты

- Получаются через сушку обычных бошек или через крафты.
- Не вызывают передозировку, дают слабые эффекты (регенерация, лёгкое замедление, голод).
- Используются для создания «Вкусного печенья» (2 печенья из 1 медицинской бошки + 2 пшеницы).

### 🔮 Прочее

- **Спайс** — синтетический наркотик, очень опасен: эффекты скорость II, голод II, прыгучесть, передозировка от 3 применений, наносит урон иссушением.
- **Удобрение** — крафтится из костной муки, гнилой плоти и измельчённой марихуаны.
- **Зелёный дым** — после употребления над игроком появляются зелёные частицы, длящиеся 10 минут (если не убрать командой `/gw smell`).

## 🧪 Рецепты (кратко)

Все рецепты зарегистрированы автоматически. Основные из них:

| Предмет | Рецепт |
|---------|--------|
| Бошка (сушка) | Соцветие → сушка → бошка |
| Мед. бошка | Бошка → сушка → мед. бошка + масло |
| Косяк | 2 бумаги + бошка (вертикально) |
| Брикет | 8 бошек в квадрате, в центре бумага |
| Пак | 9 брикетов в квадрате |
| Гашиш | Печка: бошка → гашиш |
| Гашишное масло | Печка: гашиш → масло |
| Гашишный пирог | Масло + гашиш + 3 пшеницы |
| Спайс | Светокамень + 2 измельчённой + масло |
| Бонг | Стеклянная бутылка + палка + огниво |
| Удобрение | Костная мука + гнилая плоть + измельчённая |
| Канна-брауни | 2 пшеницы + какао-бобы + сахар + масло |
| Вкусное печенье | 2 пшеницы + мед. бошка → 2 печенья |

Все предметы используют кастомные `NamespacedKey` и хранят данные в `PersistentDataContainer`.

## 🛠️ Для разработчиков

Плагин написан на Java с использованием Paper API. Основные классы:

- `GreatWeeb` — главный класс, регистрация слушателей, менеджеров.
- `StrainItemsImpl` — общая логика для Сативы и Индики.
- `RecipeManager` — регистрация всех рецептов.
- `OverdoseListener` — механика передозировки.
- `DelayedEffectManager` — отложенные эффекты.
- `SmokeEffectManager` — визуальный дым.
- `DryingRackManager` — тикание сушилок.
- Различные `*Listener` — обработка взаимодействий.

Плагин **не использует конфигурацию** — все параметры (пороги, длительности, эффекты) зашиты в коде. При желании их можно вынести в `config.yml`.
